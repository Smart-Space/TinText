# Tin标记语言（TinML）
一门简洁、清晰、规范、易转译、用于流式渲染的标记语言。

---

## 特性
TinML具有如下特性：
1. 简洁
2. 清晰
3. 规范
4. 可转译
5. 可流式渲染

---

## 发展
### 起源
2021年，Tin标记语言作为内置。。。

### 成熟
。。。

### 重构
2024年，抛弃原有框架，直接重写。

这次，Tin标记语言不再作为“tkinter渲染标记语言”中的一环，而是被定义为一门真正的标记语言，从格式、语法、具体标签、解释、转译等方面重新定义TinML。新的TinText应用只是TinML的一种实现。
