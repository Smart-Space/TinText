# TIN文件

## 简述

TIN文件为TinML的源码格式文件。

## 后缀名

`*.tin`

## 编码

UTF-8

---

## 描述

TIN文件为一种源码格式文件。

### 开头约定

TIN文件为TinML源码存储，无任何强制性开头约定。

一般地，建议在开头加上`<tin>`标签，标明这是TinML的源码文件。`<tin>`标记不会作用于最终呈现，而是会被执行TinML标准的渲染器忽略。`<tin>`后可接任意文本，建议填写文件相关信息。

```
<tin>this file is for ...
<t>TITLE

...
```

### 文本内容

TinML源码。

### 写入

按UTF-8编码写入。

### 读取

按UTF-8编码读取。
