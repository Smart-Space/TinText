# TinText

Tin标记语言渲染的python.tkinter实现。

[TinText网站](https://tintext.smart-space.com.cn/)

## Tin标记语言

一个能够实时呈现渲染的流式标记语言。

---

## 依赖

tinui - 作者本人开源 tkinter UI 库

requests

PIL(pillow)

dominate - html编辑库

html2text - markdown转译技术支持库
